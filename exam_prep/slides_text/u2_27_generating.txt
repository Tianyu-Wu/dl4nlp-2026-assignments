beam search decoding: pseudocode

set the beam width k
initialize X = x1, . . . , xm to some token sequence
B ← [X]
for i = m + 1, . . . until some stopping criterion met

C ← [ ]
for each b in B

compute P (x|b)
add b + [x] to C for all x in the vocabulary

B ← select k top-scoring candidates from C

return top-scoring beam from B
